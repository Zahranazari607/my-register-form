# Week 5 Assignment: Register Form with Validation

This project is a simple Registration Form built using **React**, **React Hook Form**, and **Yup** for validation.

## Features
- **Full Name**: Minimum 3 characters required.
- **Email**: Valid email format required.
- **Password**: Minimum 8 characters and must include at least one number.
- **Confirm Password**: Must match the entered password.
- **Terms & Conditions**: Must be checked to submit.
- **Validation**: Real-time validation using Yup resolver.

## Technologies Used
- React (Vite)
- React Hook Form
- Yup (Validation Schema)
- CSS3 (Custom styling)
